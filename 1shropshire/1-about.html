About

Overview

This Web Site is my personal project about the County of Shropshire. The Web Site explores various aspects of the county, such as its history and landscape.

GitHub Pages

The Web Site is hosted as a GitHub Pages site, this provides a GitHub respository and hosting for a site made up of static pages.

Jekyll

The static web pages are built using the Jekyll web site generator.

The source files for Jekyll are a combination of Liquid templates and YML data files.

BootStrap

The template for the Web Site is built using the BootStrap framework.
